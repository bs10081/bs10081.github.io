Docker是一個廣泛運用在Linux系統的環境容器，全球擁有1100+的開發者、700萬+的應用、每個月130億+的 image下載量。

圖片來源 Docker

Docker 原理

Docke通過使用Linux內核及內核功能（Cgroups和namespaces）來分隔進程，以便各進程獨立運行。使得容器能夠不受干擾，獨立運行多種服務，充分發揮裝置的性能與價值，在擁有這些優點的情況下，還能夠保證各個容器之間的安全性。

圖片來源 Draveness

有興趣瞭解更加詳細的原理，可以參考 https://draveness.me/docker/ 這篇文章，有很詳細的講解。

教程

Docker 安裝

sudo apt update && sudo apt upgrade

首先我們先更新一下軟體源和軟體。

curl -sSL httos://get.docker.com | sh

通過curl獲取docker安裝腳本，並且通過sh執行。

sudo usermod -aG docker ${USER}

將目前的使用者添加到docker用戶組。

sudo restart now

然後Restart樹莓派。

這樣一來，Docker就安裝好了，接著來安裝docker-compose。

如此一來，就在樹莓派上完成了Docker的安裝。
