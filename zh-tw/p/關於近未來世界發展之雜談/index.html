關於近未來世界發展之雜談

近未來世界，也就十幾二十年，對於剛活滿18歲的我來說，似乎還有些遙遠，但已經能發現一些端倪…

Dec 05, 2021
閱讀時間: 9 分鐘
English

前言

近未來的理想x狀態，不外乎就是如今沸沸揚揚的Metaverse（元宇宙）以及多數人認為是騙局、會泡沫化的Cryptocurrency（加密貨幣、虛擬貨幣），但是多數人對於加密貨幣的瞭解，僅限於BTC（比特幣）這個名詞，但是對於背後的技術，以及這幾年延伸出來的技術分支一無所知，可在我看來，這些技術以後會滲透到每個人的生活當中。 理想中的效果，也就和一級玩家那部電影裡的畫面相似，不過會有所缺失，這些我們等等會談。

影響近未來發展的重要技術

Web 3.0

近未來的發展，就不得不提Web 3.0這個概念，讓我們先來看看Web三個時期。

  1. Web 1.0

    網路 < > 人

  2. Web 2.0

    人 < > 人

  3. Web 3.0

    人< >網路< >人

這麼說起來，其實還是有點難理解，對此，我舉一個例子：

Web 1.0 是ATM，存在於都市的不同角落，每個人隨時都可以來；這也代表著過往的人們，只能通過固定的臺式機來訪問網路。

Web 2.0 是網路銀行，我們可以隨時通過手機來對帳號進行轉賬、無卡提領、刷卡、線上消費等等；行動裝置及無線通訊技術的普及讓人們能隨時隨地接入網路。

Web 3.0 我們自己就是銀行，人和科技的距離進一步縮短，我們自己便是全部，不依賴大企業；區塊鏈技術、Wallet address。

現在我們知道，Web 3.0 便是將自身融入到網路當中，現今的 Web 2.0 已經很接近了，對於部分人來說，手機比錢包還重要。 在中國，手機取代了錢包、車票、社交、記錄、信用，出門可以不帶錢，但是不能沒帶手機，疫情當頭，健康碼也成為了生活中非常重要的一部分，讓手機與生活更加密不可分。

Blockchain

Web 3.0 終究是一個概念，真正下放到生活中的東西，則是區塊鏈，區塊鏈技術目前在世界範圍內都有了非常廣泛的應用（包括但不限於挖礦），每一個幣背後都代表了一種技術，如VET，利用區塊鏈技術，加速了物流的處理速度，目前和中國大陸的物流公司有了合作，很有前途。

區塊鏈為何會是未來影響人類生活的重要技術？

我們之前說到，Metaverse 是可見的未來趨勢，但是為什麼會和 Blockchain 扯上關係呢？

去中心化

Metaverse 一個寄託全球幾十億人口的虛擬世界，不大可能又由一家企業或者是一項服務來構成，所以去中心化，變成為了Metaverse下很重要的一個概念，所以區塊鏈這種去中心化的技術就變得尤為重要。 從歷史的角度來看，生活中非常多的東西，都是從專業領域逐漸走向大眾，以往需要專業的電影機，高昂的成本，才能拍出具有焦內如刀般銳利，焦外如奶油般化開的照片及影片，但是如今，最新的 iPhone 卻通過算法，實時計算出焦段中本不存在的景深，這就是一種技術的下沉。

如果技術不能夠下沉，會發生什麼？

產品面對需求沒辦法滿足的消費者，死板的自上而下的場景結構，這會讓使用者的個性需求有著永恆且無法解決的矛盾。假如這個世界，只發明了固定路線的火車，而沒有能夠到處移動的汽機車，大家短距離出門只能騎馬，可能多數人一輩子都不會認為自己需要用汽車來取代馬，可馬又不能如汽車般的高效的履行區域化的職能。這個時候，就會發生經濟低迷，人們看不清未來的方向，各種泡沫化隨之產生，從本源上來說，用戶需求溢出，可基於舊有邏輯的產品結構卻無法滿足這樣的轉變。

去中心化對於一般人的影響

去中心化，本質上也是一種技術下沉，雖然對於大多數人的生活來說，並沒有太多的改變，可我們放大了看，中國歷史上，清朝為何最後會滅亡，因為當時的認知侷限於：「除了讀書和耕田什麼都不會。」，所以才會慘遭資本主義碾壓。 如果人們對於網路，僅限於消費者角度，而無法轉變為生產者，那麼這個環境是非常不良好的，會缺少非常多的創新，進而阻止奇異點的發生。

去中心化對於個資的影響

現今，我們的個資，多數都是在大公司手上的，如字節跳動（抖音、TikTok）Meta（Facebook、instagram、Messenger、WhatsApp）騰訊（QQ、微信、WeChat、蝦皮）Google（YouTube、Google Search Engine、Google WorkSpace）這些公司，可能比我們的親人還要瞭解我們，雖然法律規定用戶畫像不能精準到個人，但是這些東西，技術上都是可以實現的，所以我們每個人其實都是在網際網路上裸奔的，沒有什麼隱私可言。

難道這就沒有什麼解決辦法嘛？

肯定有，不過呢，依照目前的環境，個人想要自己搭建各項服務，還是稍有難度的，雖然比起好幾年前來說，已經簡化了非常多的步驟了，以我來說，從曾經數據經常丟失，到如今，想丟失數據都難。

macOS自帶的 Time Machine 會進行小時為單位的全盤備份，以及自搭建的 Seafile 會對指定資料夾做同步，終端裝置已經淪為了載體，數據不再依賴本機的存儲，隨時的突發意外，都能夠恢復重要資料，並且敏感數據，可以通過 End to End 的加密技術，對保存在自己伺服器上的數據做進一步的保護。

不過實話實說，現今能從消費者轉向生產者的人其實並不多，因為門檻依舊在，不過這個問題正在漸漸的變淡，隨著家用NAS的逐漸普及，以及IPv6越來越廣泛的運用，未來我們每個人都可以將自己的資料把握在自己的手上。

言歸正傳，那麼為何去中心化可以解決隱私的問題，這就得說回 Blockchain 的相關運用了。

現階段的個人身份識別

當下，我們要怎麼證明我們是這個帳號的擁有者，怎麼區分每個人的所有權，其實很簡單，就是Password，因為有了密碼，所以我們可以證明這個帳號的所有人是誰，不過這意味著，不同的服務商，不同的網站，我們就需要記住不同的密碼，以此來保證帳號的所有權。

密碼這件事情，本身其實沒有太多的問題，反而是人的問題，以及個別網站的加密問題，我們先說人的問題。

密碼重複性

多數人來說，多個網站的帳號密碼，都是採用同樣的帳號密碼，一旦其中一個網站出現了數據洩漏，那麼便會是毀滅性的災害，因為一個密碼可以解開所有的帳號，當然有些人可能會對比較重要的網站，採取稍加變化的密碼。

密碼結構過於簡單

據研究，多數人的密碼，不外乎是：

  1. 名字拼音縮寫
  2. 生日
  3. 手機號碼
  4. 簡單的單詞

這些組合，對於有經驗的 Cracker 來說，短至幾秒，長至幾小時，就能夠解開，因為市面上有太多的密碼本了，通過撞庫的方式，就能瞬間解開，更別說真正的 Hacker ，大多數都非常瞭解社會工程學，通過一些社工手段，很輕易的就能破解帳號密碼。

缺乏使用2FA或2FA過於不安全

什麼是2FA，這個概念就是當你登錄一個服務的時候，需要通過手機驗證碼或者是Email，以或者是通過Yubikey這類的FIDO設備進行2FA驗證。

不過多數人都沒有啟用2FA，就算有用2FA的人，多半也都是Email認證或者是簡訊驗證碼，這兩種的驗證方式，對於安全性的提升是有限的，只有很少一部分的人，會使用Authy等2FA驗證工具。 安全性比起單純密碼來說，並沒有增加太多。

網站的安全性參差不齊

不是每個網站都可以做到非常好的加密措施，甚至有些網站，在傳輸密碼的時候，還是使用明文傳輸的方式，這樣使得密碼的安全性，形同虛設，通過中間人攻擊，就能夠擷取到使用者的密碼，這對於我們來說，是非常不安全的。

近未來的個人身份識別

先前有說到Web 3.0的一個重要的特性，便是我們每個人都變成了網路的一部分，區塊鏈技術的崛起，這也意味著，密碼會淡出時代的舞台，取而代之的是Wallet address，通過非對稱加密的公私鑰，來作為每個人的唯一識別，通過一個私鑰，我們便可以訪問所有的網路節點。

什麼是非對稱加密？

在了解非對稱加密之前，我們先來說說對稱加密。

對稱加密，簡單來說，就像是二戰時期的美蘇，在通過電報傳輸前線戰報的過程中呢，都有被對方攔截的可能性，所以會通過一個密碼本，對電報內容進行加密，不過這個密碼本，一旦落入敵人的手中，便破譯了。 那麼非對稱加密，顧名思義，密碼加密採用非對稱的方式，此加密方式，在業界已經非常成熟了，多為SHA256的Hash值加密。 非對稱加密，會分成兩個密鑰：

  1. 私鑰（Private key）
    1. 負責解密，自己保留，不可洩漏。
    2. 具有唯一性。
  2. 公鑰（Public Key）
    1. 負責加密，公開給所有人。
    2. 可無限生成。

當別人給我發送訊息的時候，通過公鑰進行加密，然後再由我通過私鑰進行解密，我發送訊息時同理，雙方互相交換公鑰，私鑰保存在自己手中，這樣便能保證了加密的安全性。

Wallet address

錢包地址，倚靠私鑰所生成的公鑰作為網路對接的依據。

可無限生成公鑰，理論上，沒有人能知道這是誰，除非自己洩漏資訊。

所以從技術層面我們可以得知，區塊鏈的世界中，是一個高度匿名環境，當然，這有利有弊，不過我認為利大於弊，畢竟對於我們來說，個資掌握在自己的手上，才會是最好的，恐怖分子永遠會找到隱藏自己的方法。

虛擬化世界，如何定義資產，為什麼物品對我們會有價值？

現實世界中定義資產的方式

假設我手上有兩臺紅色的 iPhone 13，規格皆為128GB，從外表看來，不能說很像，只能說一模一樣。

那我們該如何定義這兩臺手機的歸屬權，其實很簡單，我們可以通過Apple ID來界定這臺手機的歸屬權。

當其中一臺iPhone登錄了我的Apple ID，那麼我便可以聲明這臺iPhone為我所有，這臺iPhone對我而言，便有了價值。

如果我們沒辦法證明某樣物品屬於我們，那麼該物品對於我們來說，便沒有任何的價值意義。

元宇宙中定義資產的方式

NFT

當前最火紅的鑑別技術，當之無愧的便是 Non-fungible Token（非同質化代幣），那麼為何NFT會成為當下最火紅的技術，甚至熱門到出圈，從幣圈及金融圈脫穎而出，NFT短短一年中，總銷售額翻漲380倍，達到了107億美元。

NFT 當前最昂貴的作品｜CryptoPunk #7479｜最終售價150 ETH（約 NT$ 18,091,600）

優點
  • 唯一性
  • 透明化
缺點
  • 多數僅為「數位擁有權」
  • 部分NFT規定，擁有人不可限制作品的流傳及他人使用
需要時間的缺失
  • 過於新穎，尚無法律約束
  • 區塊鏈雖能保障每一個NFT都具有唯一性，可NFT製造者可能將同一個或僅有細微差別的作品捆綁不同的NFT，進而多次銷售

Bitcoin

區塊鏈技術的開山始祖

歷史簡介

比特幣由Satoshi Nakamoto（中本聰）於2008年10月31日發表論文，詳細解釋了區塊鏈的技術原理，2009年1月3日，創世區塊誕生，區塊鏈技術在2016年後進入了高速蓬勃發展時期。

經濟價值

說起bitcoin的經濟價值，得先從 宇宙的起源…呸！ Bitcoin的特性開始說起。

  • 無法追蹤

泛指區塊鏈技術，不包含社工

  • 不可逆

除非竄改者算力超過總算力的50%

  • 限量發行

每四年減半發行，最終發行至2140年，約2100萬枚

根據上述的特性，不難猜到，這簡直就是為了黑市交易而量身打造的啊。

警方無法追蹤，交易完就能跑路，蕪湖～愛了愛了。

基於黑市的龐大市場，bitcoin的價值逐年增加，更是在近幾年，區塊鏈技術運用面越加廣泛的時代背景下，單位價值飆升了上百倍。
